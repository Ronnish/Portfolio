# CreativePress #
Clean Magazine Theme with dynamic color. Create amazing magazine or blog site using widgets and various options. Get Free Support at http://nextwp.com

CreativePress WordPress Theme, Copyright (c) 2017, NextWP
CreativePress is distributed under the terms of the GNU GPL

# Installation #
1. After downloading the zip file, go to Appearance > Themes and click on the *Install Themes* tab.
2. Click on the *Upload* link.
3. Upload the zip file that you downloaded and click *Install Now*.
4. Click *Activate* to use the theme you just installed.
